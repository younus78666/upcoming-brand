## Publishing Rules (Apply to Every Post)

### On-Page SEO (Every Post)
- H1 = primary keyword (exact or close variation)
- Meta title = primary keyword + "| Upcoming Brand" (≤60 chars)
- Meta description = keyword + USP + CTA (≤155 chars)
- URL = /blog/[primary-keyword-slug]/
- H2s: all end with ? (UC v6.4 rule)
- FAQPage schema: minimum 4 FAQ pairs per post
- Article schema: with author, datePublished, dateModified
- Word count: 1,200–2,000 words for informational; 800–1,200 for comparison

### Internal Linking Rules (Every Post)
1. Link to 1 SERVICE PAGE (most relevant to post topic)
2. Link to 1 LOCATION PAGE (most relevant city)
3. Link to 2–3 BLOG POSTS within same cluster (when available)
4. Link to /pricing/ when cost is mentioned
5. Link to /tools/installment-calculator/ when payment is mentioned
6. Anchor text = descriptive keyword phrase, never "click here"

### Content Structure (Every Post)
- Intro: Hook + direct answer (AI Overview snippet target) in first 100 words
- H2 sections: 3–6 sections, each 150–300 words
- CTA block: mid-post + end-post (link to relevant service page + /book-a-discovery-call/)
- FAQ section: 4–6 questions, schema-marked
